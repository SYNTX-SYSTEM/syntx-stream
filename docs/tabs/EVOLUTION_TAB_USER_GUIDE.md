Was ist Maturity?

Ist das Echtzeit?
- Daten beim Laden geholt
- F5 fuer Refresh

Unterschied zu normalen Dashboards?
- Normal: Zeigt WAS passiert (Zahlen)
- SYNTX: Zeigt WIE ES SICH ANFUEHLT (Emotion)

Desktop vs Mobile?
- Desktop: Best Experience, Hover funktioniert
- Mobile: Components stacken, Touch statt Hover
